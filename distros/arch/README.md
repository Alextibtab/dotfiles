# My Arch Linux Setup

This repository contains my personal Arch Linux configuration files (dotfiles) for a minimal and modern desktop environment.

## Core Components

- **Display Server**: Wayland
- **Window Manager**: Hyprland
- **Display Manager**: SDDM
- **Boot Manager**: rEFInd

## Desktop Environment

- **Status Bar**: Waybar
- **Application Launcher**: Wofi
- **Notifications**: Dunst
- **Terminal**: Alacritty
- **Shell**: Nushell with Starship prompt
- **Lockscreen**: Hyprlock
- **Logout Manager**: wlogout
- **Wallpaper Handling**: SWWW + Waypaper

## Features

- Modern Wayland-based setup
- GPU-accelerated animations with Hyprland
- Custom terminal experience with Alacritty + Nushell + Starship
- Clean system management with SDDM and rEFInd
- Dynamic wallpaper management with SWWW and Waypaper

## Components Overview

### Core System
- **Wayland**: Modern display server protocol
- **Hyprland**: Highly customizable dynamic tiling Wayland compositor
- **SDDM**: Simple Desktop Display Manager
- **rEFInd**: Beautiful and powerful boot manager

### User Interface
- **Waybar**: Highly customizable status bar
- **Wofi**: Application launcher and menu
- **Dunst**: Lightweight notification daemon
- **Hyprlock**: Lock screen solution for Hyprland
- **wlogout**: Logout menu for Wayland
- **SWWW**: Efficient wallpaper daemon for Wayland
- **Waypaper**: Wallpaper management tool

### Terminal Environment
- **Alacritty**: Modern, GPU-accelerated terminal emulator
- **Nushell**: Modern shell with structured data support
- **Starship**: Minimal and customizable shell prompt
