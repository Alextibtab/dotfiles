# Catppuccin Peach Theme

This theme is based on catppuccin and uses the peach colour as the accent for most applications where possible.

## Core Packages

| Package   | Description        |
|-----------|-------------------:|
| Hyprland  | Window Manager     |
| Hyprlock  | Screen lock        |
| Hyprshot  | Screenshots        |
| Hyprcursor| Cursor             |
| alacritty | Terminal           |
| nushell   | Shell              |
| starship  | Prompt theme       |
| nvim      | IDE                |
| fastfetch | Neofetch           |
| waybar    | Menu bar           |
| wofi      | Application runner |
| swww      | Wallpaper daemon   |
| waypaper  | Wallpaper picker   |
| sddm      | Display manager    |
| wlogout   | Logout menu        |
| spicetify | Spotify themeing   |
| thunar    | File browser       |
